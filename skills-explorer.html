Installing Skills

Skills work across Claude Code, Claude.ai, and the Claude API. Install from the marketplace or reference the GitHub repo directly.

Claude Code — Marketplace
/plugin marketplace add anthropics/skills
/plugin install <skill-name>
Claude Code — Direct
claude plugin install \
  anthropics/skills/skills/<skill>
Claude.ai
Settings → Skills → Upload
Upload your SKILL.md file directly
(Paid plans)
GitHub Repo
git clone https://github.com/
  anthropics/skills
cp -r skills/<skill> .claude/skills/
